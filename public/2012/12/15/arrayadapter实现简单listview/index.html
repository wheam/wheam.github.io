其实ListView一直没有真正实际运用到程序中过（大部分写的都比较奇葩，非信息呈现类），看了很多软件，尤其面包旅行这种，感觉ListView实现大图片应用也是十分好看的，嗯呢。

034     @Override

035     public void onCreate(Bundle savedInstanceState) {

037         setContentView(R.layout.activity_main);

038         listView=(ListView)findViewById(R.id.list);//获取布局文件中已经写好的ListView

040         listString.add("这是第一个");//这个就是单纯为了保证ListView里面开始不会什么都没有

041         adapter=new ArrayAdapter<String>(this, android.R.layout.simple_list_item_1,listString);

042         //new ArrayAdapter<String>(context, textViewResourceId, objects)中的三个参数

043         //context 官方文档解释是The current context. 一直想不通，看到大家用this就跟着用了，求大神解释

044         //textViewResourceId是指ListView中每一个Item的布局，我代码里使用的是默认的单行文字布局，没有自定义，可以自定义

045         //最后的objects就是你的数据，一个String数组或者其他数据容器，大部分示例代码中会使用一个自定义getData（）方法来返回一个设定好的ArrayList<String>，我这里直接使用的自定义好的

046         

047         listView.setAdapter(adapter);//设置适配器
